# Resume

AI-powered resume builder by Festus Kahunla.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Zustand (state management)
- @react-pdf/renderer (PDF generation)
- Multi-provider AI (Claude, GPT-4o, Gemini, DeepSeek)

## Acknowledgements

This project was inspired by [OpenResume](https://github.com/xitanggg/open-resume), an excellent open-source resume builder that I've used extensively to create resumes for friends and many others. It's a great piece of software built by talented people. Resume builds on that foundation by adding AI-powered features for a faster, more guided resume-building experience.
